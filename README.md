## Getting Started
Install the module with: `npm install cena-auth`

```javascript
var cenaAuth = require('cena-auth');
cenaAuth.awesome(); // "awesome"
```

```coffee-script
cenaAuth = require 'cena-auth'
cenaAuth.awesome() // "awesome"
```
